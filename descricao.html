Descrição

Projeto e implementação de uma interface entre um sistema de reconhecimento de voz e um robô móvel.
O presente artigo descreve o funcionamento do robô móvel que interage com o usuário a partir de uma interface dotada de reconhecimento de voz, a interação pode ocorrer com o robô estando no mesmo ambiente que o usuário ou distancia a partir de um sistema web de controle. Utilizamos uma plataforma de código aberto para o desenvolvimento desse projeto, o Jasper, que disponibiliza ferramentas para aplicações utilizando o reconhecimento de voz, o computador responsável pelo processamento dos dados é uma rasberry PI modelo 3 B.
